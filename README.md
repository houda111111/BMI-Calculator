# BMI Calculator

This is a simple and modern BMI Calculator project that I built using HTML, SCSS, and JavaScript.

The project calculates the user's BMI (Body Mass Index) based on height and weight, then displays the weight condition.

## Features

- Calculate BMI
- Modern responsive design
- Glassmorphism UI
- Show weight condition:
  - Underweight
  - Normal Weight
  - Overweight
  - Obesity
- Save data using Session Storage

## Technologies Used

- HTML5
- SCSS
- JavaScript
- Git & GitHub
